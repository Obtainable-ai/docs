The PeerNotes browser extension is currently available as a pre-release build. It has not yet been published to the Chrome Web Store, so you will install it manually by loading the unpacked extension through Chrome's Developer mode. This takes less than a minute and requires no special permissions beyond enabling Developer mode in your browser.

<Warning>
  This installation process is temporary. Once the PeerNotes extension is published to the Chrome Web Store, you will be able to install it in one click and receive automatic updates. Manual unpacked installs will no longer be necessary at that point.
</Warning>

## Installation steps

<Steps>
  <Step title="Extract the ZIP file">
    Locate the downloaded PeerNotes ZIP file and extract it to a permanent folder on your computer. Avoid placing it in a temporary location — Chrome needs access to this folder every time you start the browser.
  </Step>
  <Step title="Open the Extensions page">
    Open Chrome and navigate to `chrome://extensions/` in the address bar.
  </Step>
  <Step title="Enable Developer mode">
    In the top-right corner of the Extensions page, toggle on **Developer mode**.
  </Step>
  <Step title="Load the unpacked extension">
    Click **Load unpacked**. A file picker will open.
  </Step>
  <Step title="Select the extension folder">
    Navigate to and select the folder you extracted in step 1. Click **Select Folder** (or **Open** on macOS).
  </Step>
  <Step title="Confirm the installation">
    The PeerNotes extension will appear in your Extensions list. It is now installed and ready to use.
  </Step>
</Steps>

## Tips

<Tip>
  **Pin the extension for quick access.** Click the puzzle-piece icon in the Chrome toolbar, find PeerNotes in the list, and click the pin icon so it stays visible in your toolbar at all times.
</Tip>

<Tip>
  **Managing the extension.** Return to `chrome://extensions` at any time to enable, disable, or remove the PeerNotes extension.
</Tip>

<Tip>
  **Updating to a newer version.** Download the latest ZIP file, extract it to replace the existing folder, then go to `chrome://extensions` and click the reload icon on the PeerNotes card to apply the update.
</Tip>
