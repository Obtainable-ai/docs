## Chrome Extension Setup

The Chrome extension is available as a pre-release build. Follow the instructions below to load it into your browser. This process will change once we deploy to the Chrome Web Store.

### Google Chrome

1. Extract the downloaded ZIP file to a folder
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top-right corner)
4. Click "Load unpacked"
5. Select the extracted extension folder
6. The extension is now installed and ready to use

### Microsoft Edge

1. Extract the downloaded ZIP file to a folder

#### Tips:

- After installing, click the extension icon in your toolbar to open the popup
- Visit `chrome(edge,brave,etc)://extensions` to manage the extension
- For updates, re-download the latest version and reload the extension
